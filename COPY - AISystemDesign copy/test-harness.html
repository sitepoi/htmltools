🔄 Agent Workflow Patterns
👤 Doer Agent
Performs actual work: drafts documents, runs queries, extracts data. Receives tasks from orchestrator, executes, submits for review.
🔍 Reviewer Agent
Reviews Doer output against rules, policies, quality standards. Flags issues, requests revisions, approves for manager sign-off.
👔 Manager Agent
Orchestrates workflow: assigns tasks to Doers, collects Reviewer feedback, makes final decisions, handles human escalations.
